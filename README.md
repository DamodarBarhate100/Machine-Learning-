
##  The ML Engine Room: Algorithms from Scratch

### *A Daily Lab for Mathematical Derivations and Implementations*
This repository is a dedicated record of my daily practice and deep dive into Machine Learning. Instead of just importing models, I decompose them. Every algorithm here is implemented via a **Dual-Track Approach**:

**1.Track 1: Mathematical Derivation:** Before a single line of code is written, I derive the cost functions, gradients, and update rules. This includes proofs for convergence and exploration of the underlying probability distributions.

**2.Track 2: NumPy (First Principles):** Translating those derivations into vectorized Python code. By avoiding high-level libraries, I master the computational logic and optimization techniques (e.g., Gradient Descent variants).

**3.Track 3: Scikit-Learn (Industry Standard):** Implementing the same model using professional frameworks to understand API design, hyperparameter tuning, and performance benchmarking.


### Tech Stack & Core Mathematics

* **Programming Languages:** Python (Advanced NumPy/Pandas)
* **Visualization:** Matplotlib, Seaborn
* **Core Math Focus:** * **Calculus:** Partial derivatives and Chain Rule for backpropagation.
* **Linear Algebra:** Matrix multiplication, Transposition, and Vectorization.
* **Probability:** Maximum Likelihood Estimation (MLE) and Distributions.
